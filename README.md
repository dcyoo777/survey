# SURVEY FORM

React, Redux 를 사용한 설문조사 양식입니다.

## 실행 방법

### `npm install`

npm install 명령어를 실행하여 필요한 패키지를 설치합니다.

### `npm start`

npm start 명령어를 실행하여 개발 서버를 실행합니다.

## 추가 구현 내용

### 양식 지우기

- 양식 지우기 클릭시 작성항 답변이 초기화됩니다.

### 데이터 저장

- 모든 데이터는 로컬 스토리지에 저장됩니다.

### 렌더링 최적화

- 불필요한 렌더링을 최소화하기 위해 useCallback 을 사용하여 최적화를 진행하였습니다.
